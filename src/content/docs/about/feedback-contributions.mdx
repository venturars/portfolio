---
title: Feedback & Contributions
---

Thank you for taking the time to read my portfolio.

If you notice something incorrect, outdated, or that could be improved, I would really appreciate your feedback. I've tried to take great care with both the content and presentation, but there may still be mistakes or things that could be improved.

If you spot anything, feel free to open an issue or suggest a pull request.

### Reporting Issues

If you'd like to report something or share feedback, you can open an issue here:

https://github.com/venturars/portfolio/issues

Even small corrections (typos, wording, missing context) are very welcome.

### Sending a Fix or Improvement

If you'd like to suggest a change directly, you can open a pull request.

Steps to contribute:

1. Clone the repository
2. Make your changes locally
3. Submit the pull request **against the `develop` branch**

Repository:
https://github.com/venturars/portfolio

#### Contribution Guidelines

Just a couple of simple things to keep contributions consistent:

- Follow the existing writing style and structure
- Keep changes focused and easy to review
- Use conventional commit messages

Thanks for taking the time to help improve the project — it's genuinely appreciated.
